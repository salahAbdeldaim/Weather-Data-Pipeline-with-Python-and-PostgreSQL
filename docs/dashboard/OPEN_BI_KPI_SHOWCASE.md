## 🛠️ How to Embed via Metabase Web Components (`SDK / <metabase-question>`)

If you are building a modern React, Vue, or Web Component-based frontend, you can use our live SDK endpoint by passing the question URL attribute directly inside `<metabase-question>`:

```html
<!-- 1. Include the Metabase Bootstrap SDK Script -->
<script defer src="http://72.62.92.93:3000/app/embed.js"></script>
<script>
  window.metabaseConfig = {
    "theme": { "preset": "light" },
    "isGuest": true,
    "instanceUrl": "http://72.62.92.93:3000"
  };
</script>

<!-- 2. Embed Specific KPI Cards or Charts by passing the 'url' attribute -->
<div class="kpi-container">
  <!-- Average Temperature KPI Card -->
  <metabase-question url="http://72.62.92.93:3000/public/question/359d33dc-ceb9-4be2-a991-8d2f3f41e127" with-title="true" with-downloads="true"></metabase-question>

  <!-- Top 10 Hottest Provinces Bar Chart -->
  <metabase-question url="http://72.62.92.93:3000/public/question/43b871a4-5d06-440f-972b-d9a0c01d080a" with-title="true" with-downloads="true"></metabase-question>
</div>
```
